이 페이지에서는 일반적으로 앱 시작 시 한 번 설정하는 **SDK 전역 기본값**을 다룹니다. 예를 들어 기본 OpenAI 클라이언트, 전송 방식, 트레이싱 내보내기 키, 디버그 로깅 동작 등이 있습니다. 이러한 설정은 기본적으로 프로세스 전역에 적용되므로, 에이전트별 또는 실행별 튜닝보다 공통 설정에 적합한 위치입니다.

### 디버그 로깅

SDK 는 디버그 로깅을 위해 [`debug`](https://www.npmjs.com/package/debug) 패키지를 사용합니다. 자세한 로그를 보려면 `DEBUG` 환경 변수를 `openai-agents*`로 설정하세요.

```bash
export DEBUG=openai-agents*
```

세션 영속성 활동을 로깅하려면 `OPENAI_AGENTS__DEBUG_SAVE_SESSION=1`을 설정하세요.

`@openai/agents`의 `getLogger(namespace)`를 사용해 자체 모듈용 네임스페이스 로거를 가져올 수 있습니다.

<Code lang="typescript" code={getLoggerExample} title="Get logger" />

#### 로그의 민감한 데이터

일부 로그에는 사용자 데이터가 포함될 수 있습니다. 다음 환경 변수를 설정해 비활성화하세요.

LLM 입력 및 출력을 로깅하지 않으려면 다음을 설정하세요.

```bash
export OPENAI_AGENTS_DONT_LOG_MODEL_DATA=1
```

도구 입력 및 출력을 로깅하지 않으려면 다음을 설정하세요.

```bash
export OPENAI_AGENTS_DONT_LOG_TOOL_DATA=1
```
